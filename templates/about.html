About

This Face Recognition and Gender Classifier was created using a sample of 14,000 images consisting of male and female faces. For each image, the face was cropped using Haar Cascade Classifier, a machine learning object detection program that identifies objects in an image and video.

After the face extraction process, each image was converted into grayscale and then resized into a 100x100 pixel image, thus totalling 10,000 pixels per image.

The next step was to reduct the amount of pixels per image with Principal Component Analysis (PCA), a method that reduces the dimensionality of the data while minimizing information loss. With 100 principal components, we were able to retain a little above 85% of the variance of the original grayscale images.

Afterwards, the PCA data was normalized into a 0-1 range and trained/tested on a Support Vector Machine classifier. After the initial performance metrics of the model were obtained, the model was hyperparameter-tuned using GridSearchCV to create a model with high test accuracy.

Finally, a model pipeline was created for new images that converts the image into the proper format for the classifier and outputs a result.
